# Email Spam Detection System

## Project Overview

This project uses Machine Learning and Natural Language Processing (NLP) to classify emails as Spam or Ham (Not Spam).

## Features

* Spam Email Detection
* Text Classification
* TF-IDF Vectorization
* Multinomial Naive Bayes Model

## Technologies Used

* Python
* Pandas
* Scikit-learn
* NLP
* Machine Learning

## Dataset

The dataset contains email messages labeled as:

* Ham (Not Spam)
* Spam

## Model

* TF-IDF Vectorizer
* Multinomial Naive Bayes

## Results

The model successfully classifies emails into spam and non-spam categories.
